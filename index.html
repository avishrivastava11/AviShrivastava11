Intro

What I am all about.

I, Avi Shrivastava, am a second year undergraduate from Birla Institute of Technology and Science, Pilani, Goa Campus. I am majoring in the field of Mechanical Engineering, expecting to pass out in 2021. I am an enthusiast in the field of Applied Mathematics, a programmer with proficiency in languages like Python, C, and C++. I am also an avid web developer with a profound knowledge of languages like Javascript, HTML, CSS, PHP, and MySQL. With this, I am also an Open Source Contributor, currently contributing to SymPy with the aim of GSoC 2019 in mind.

I started contributing to open source in June 2018, as a project of QSTP - Intro to Python course, which I took to improve my skills in the mentioned language. Since then, I have been active in SymPy's contribution, getting 2 PR's merged.

Working under the QSTP - Python course, I made some independent projects, out of personal interest and in attempt to improve my skills like Space Invaders Game, Snake Game, 2048 game - console, 2048 Game - PyGame Version 1, 2048 Game - PyGame Version 2, Sudoku Game - Console, Tic-Tac-Toe Game - Console - Beatable, Tic-Tac-Toe Game - Console - Unbeatable, and many more..
